    X-R Stats

    1. Mean (X-bar) Chart

    The X-bar chart monitors the mean of subgroups over time. The UCL and LCL are derived from the average of these means and the process variability.

    • Mean (X-bar):

      X̅ = (Sum of the means of all subgroups) / (Number of subgroups)

    • Upper Control Limit (UCL) for X-bar:

      UCL(X) = X̅ + A₂ × R̅

      Where:

      • X̅ = Average of subgroup means

      • A₂ = Control chart constant (depends on subgroup size)

      • R̅ = Mean range

    • Lower Control Limit (LCL) for X-bar:

      LCL(X) = X̅ - A₂ × R̅

    The constant ( A2 ) is found from standard statistical tables based on subgroup size:

    Subgroup Size (n) ( A2 )
    2 1.880
    3 1.023
    4 0.729
    5 0.577
    6 0.483

    Given:

    • X̅ = 10.00

    • R̅ = 0.05333

    • Subgroup size n = 5 (so A₂ = 0.577)

    Upper Control Limit (UCL) for X-bar: UCL(X) = X̅ + A₂ × R̅
    UCL(X) = 10.00 + 0.577 × 0.05333 = 10.03

    Lower Control Limit (LCL) for X-bar: LCL(X) = X̅ - A₂ × R̅
    LCL(X) = 10.00 - 0.577 × 0.05333 = 9.97

    2. Range (R) Chart

    The R chart monitors the variation (range) within your subgroups. The control limits depend on the average range and control chart constants.

    • Mean Range (R-bar):

      R̅ = (Sum of the ranges of all subgroups) / (Number of subgroups)

    • Upper Control Limit (UCL) for R chart:

      UCL(R) = D₄ × R̅

    • Lower Control Limit (LCL) for R chart:

      LCL(R) = D₃ × R̅

    If LCL(R) is negative, set it to 0.

    The constants ( D3 ) and ( D4 ) depend on subgroup size.

    Subgroup Size (n) ( D3 ) ( D4 )
    2 0.000 3.267
    3 0.000 2.574
    4 0.000 2.282
    5 0.076 2.114
    6 0.136 2.004

    Example:

    If R̅ = 0.05333 and subgroup size n = 5:

    Upper Control Limit (UCL) for R chart:

    UCL(R) = 2.114 × 0.05333 = 0.11277 (rounded to 0.11)

    Lower Control Limit (LCL) for R chart:

    LCL(R) = 0.076 × 0.05333 = 0.00405 (rounded to 0.00)

    3. Process Capability Indices (Cp and Cpk)

    These indices measure how well the process is performing relative to the specification limits.

    • Cp (Process Capability Index):
       Cp = (USL - LSL) / (6 × σ)

    Where:

    • USL = Upper Specification Limit

    • LSL = Lower Specification Limit

    • σ = Process standard deviation (estimated using R/d₂, where d₂ is a constant depending on subgroup size)

    • Cpk (Process Capability Index):

      Cpk = min [(USL - X̅) / (3 × σ), (X̅ - LSL) / (3 × σ)]

    This compares how centered the process is between the specification limits.

    • Constants ( d2 ) for subgroup sizes:
    Subgroup Size (n) ( d2 )
    2 1.128
    3 1.693
    4 2.059
    5 2.326
    6 2.534

    Example:

    Given:

    • USL = 10.05

    • LSL = 9.95

    • X̅ = 10.00

    • σ = 0.05333 / 2.326 = 0.02293

    Calculate Cp:

    Cp = (USL - LSL) / (6 × σ)

    Cp = (10.05 - 9.95) / (6 × 0.02293)

    Cp = 0.1 / 0.13758 = 0.727

    Calculate Cpk:

    Cpk = min [(USL - X̅) / (3 × σ), (X̅ - LSL) / (3 × σ)]

    Cpk = min [(10.05 - 10.00) / (3 × 0.02293), (10.00 - 9.95) / (3 × 0.02293)]

    Cpk = min [0.05 / 0.06879, 0.05 / 0.06879]

    Cpk = 0.727

    Summary of Steps

    1. Calculate the Mean (X̅) and Range (R̅).
    2. Use constants ( A2 ), ( D3 ), and ( D4 ) based on your subgroup size.
    3. Calculate the Control Limits for both the X-bar chart and the R chart.
    4. Use process standard deviation ( σ) (estimated using (R̅/ d2 ) to calculate Cp and Cpk.
